[role="lead"]
此資料收集器支援從NetApp Cloud Volumes Services收集AWS組態的庫存。



== 組態

[cols="2*"]
|===
| 欄位 | 說明 


| 雲端Volume區域 | 適用於AWS的NetApp Cloud Volumes Services區域 


| API金鑰 | Cloud Volumes API金鑰 


| 秘密金鑰 | Cloud Volumes秘密金鑰 
|===


== 進階組態

[cols="2*"]
|===
| 欄位 | 說明 


| 庫存輪詢時間間隔（分鐘） | 預設為60分鐘 
|===
