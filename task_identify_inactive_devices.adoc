[role="lead"]
識別您擁有的資產及使用者是「適當調整規模」及釋出未使用基礎架構的關鍵。您可以輕鬆地重新分配或取消分配使用不足的資源、避免不必要的採購。

請使用下列步驟來識別非使用中資產。

.步驟
* 瀏覽至 * 瀏覽 -> + 新指標查詢 * 。
* 從下拉式清單中選取 _Storage_ 。
* 按一下齒輪、並將 _IsActive_ 新增為欄。
+
顯示檢查的列為作用中。「 x 」表示非使用中的裝置。
