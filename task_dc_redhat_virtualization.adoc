= Red Hat 虛擬化資料收集器
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
用於 Red Hat Virtualization 的 DII 資料收集器也支援開源專案 oVirt、以及 Oracle Linux Virtualization Manager。此資料收集器透過 REST API 與管理站上的 HTTPS 介面通訊。



== 術語

對於Data Infrastructure Insights所取得的每種資產類型，都會顯示該資產最常用的術語。當查看或排除此資料收集器的故障時，請記住以下術語：

[cols="2*"]
|===
| 供應商/型號術語 | Data Infrastructure Insights術語 


| 磁碟 | 虛擬磁碟 


| 主持人 | 主持人 


| 虛擬機 | 虛擬機 


| 儲存域 | 資料儲存 


| 邏輯單元 | 邏輯單元號 
|===
注意：這些只是常見的術語映射，可能並不代表此資料收集器的所有情況。



== 要求

* 透過 REST API 經由連接埠 443 取得 RHV/oVirt/OLVM 伺服器的 IP 位址/主機名稱
* 只讀使用者名稱和密碼
* RHV 版本 3.0+ / Ovirt 4.0+ / OLVM 4.0+




== 配置

[cols="2*"]
|===
| 場地 | 描述 


| RHV/Ovirt/OLVM 伺服器 IP 位址 | 管理站的 IP 位址 


| 使用者名稱 | 使用者名稱 


| 密碼 | 密碼 
|===


== 進階配置

[cols="2*"]
|===
| 場地 | 描述 


| HTTPS 通訊埠 | 用於與 RHEV 進行 HTTPS 通訊的端口 


| 庫存輪詢間隔（分鐘） | 預設值為 20 分鐘。 


| 虛擬機器效能 | 啟用效能收集 


| 效能輪詢間隔 （秒） | 預設值為 900 
|===
