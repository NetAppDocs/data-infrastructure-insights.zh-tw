== 安裝

有關 Kubernetes 安裝說明，請參閱link:task_config_telegraf_agent_k8s.html["Kubernetes 監控操作員"]頁。



== 關於 Kube-state-metrics

Kubernetes Monitoring Operator 自動安裝 kube-state-metrics；無需使用者互動。
