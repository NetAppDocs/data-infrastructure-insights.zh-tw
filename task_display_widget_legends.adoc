= 顯示小工具圖例
:hardbreaks:
:allow-uri-read: 


[role="lead"]
儀表板上的視窗小工具可以顯示圖例，也可以不顯示。

顯示儀表板時，可以透過點擊小工具上的「*選項*」按鈕並在選單中選擇「*顯示圖例*」來開啟或關閉小工具中的圖例。

隨著小部件中顯示的資料變化，該小部件的圖例也會動態更新。

當顯示圖例時，如果可以導覽至圖例指示的資產的登入頁面，則圖例將顯示為指向該資產頁面的連結。
