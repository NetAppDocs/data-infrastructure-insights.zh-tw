= NetApp通用資料庫主機解析器
:hardbreaks:
:allow-uri-read: 


[role="lead"]
此數據收集器獲取庫存



== 術語

Data Infrastructure Insights從該收集器獲取庫存。



== 配置

[cols="2*"]
|===
| 場地 | 描述 


| 資料庫伺服器 IP 位址 | 資料庫伺服器的IP位址 


| DB 伺服器連接埠 | 預設值為 3306 


| 合格的驅動程式類別名稱 | 驅動程式的類別名稱。 


| JDBC 驅動程式 URL | 驅動程式的 URL 


| SQL查詢 | 用於從表中傳回資料的查詢 
|===


== 進階配置

[cols="2*"]
|===
| 場地 | 描述 


| 庫存輪詢間隔（分鐘） | 預設為 60 分鐘 


| 啟用 SSL 連接 | 選擇啟用透過 SSL 的連接 


| 報告 WWN 別名 | 選擇報告 WWN 別名 
|===
