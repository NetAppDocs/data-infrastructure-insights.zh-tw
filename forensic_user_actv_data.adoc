= 取證用戶活動數據
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
使用者活動頁面可協助您了解使用者對實體執行的操作。

本頁提供的數據顯示：

* 實體存取時間，包括月、日、年、時間。
* 透過連結存取實體的用戶link:forensic_user_overview.html["用戶詳細資訊數據"]。
* 使用者執行的活動：
+
** 寫
** 讀
** 存取元數據


* 指向實體的路徑，其中包含指向link:forensic_entity_detail.html["實體詳細數據"]
* 實體類型：
+
** 文件
** 小路


* 執行操作的 IP 位址。
